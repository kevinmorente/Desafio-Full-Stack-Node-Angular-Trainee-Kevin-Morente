Bem-vindo ao meu projeto de Desafio Full Stack! Neste desafio, desenvolvi uma aplicação de lista de tarefas (To-Do-List) usando um backend em Node.js e um frontend em Angular. A aplicação utiliza autenticação JWT para segurança, e o banco de dados é gerenciado com SQLite. A comunicação entre o frontend e o backend é feita por meio de HTTPClient.

Estou animado para apresentar como integrei diferentes tecnologias, como JavaScript, Node.js, Angular, e SQLite. Espero que este projeto demonstre meu comprometimento com a qualidade e inovação em desenvolvimento.
